## RSink
Simple utility to backup/sync data between devices to the cloud


### Supported Platforms
- [X] Linux
- [ ] Mac (Not tested)
- [ ] Windows (Not tested)
- [X] Android (via [termux](https://github.com/termux/termux-app) + [termux boot](https://wiki.termux.com/wiki/Termux:Boot))

### Supported Providers
- [X] S3 Storage
- [ ] Google Drive
- [ ] One Drive
- [ ] Proton Drive
- [ ] Dropbox
- [ ] Mega
